中国—乌兹别克斯坦经贸合作分委会第九次会议在塔什干召开
4 月 15 日，商务部副部长兼国际贸易谈判副代表凌激在塔什干与乌兹别克斯坦投资、工业和贸易部部长库德拉托夫会谈，并与副部长卡西莫夫共同主持召开中国—乌兹别克斯坦经贸合作分委会第九次会议。凌激表示，在中乌元首战略引领下，双边经贸关系持续健康发展，双边贸易快速增长，产供链合作稳步发展。双方将优化贸易结构，培育跨境电商、数字贸易等新业态；用好各类投融资平台，深化绿色矿产、基础设施建设、经贸园区等领域合作；推动中国—中亚贸易投资自由化便利化，促进区域经济合作高质量发展。中方坚定支持乌兹别克斯坦加入世贸组织。乌方表示，中国是乌最大贸易伙伴和外资来源国，对华经贸合作对促进乌现代化进程和民生改善意义重大。愿与中方进一步加强光伏发电、汽车组装、信息通信、数字经济等领域合作，深化金融机构间交流，密切地方经贸往来，并为中国企业赴乌投资兴业创造良好营商环境。感谢中方对乌方申请加入世界贸易组织的宝贵支持。会后，双方共同签署会议纪要。

算力涨价潮席卷全球：阿里、Anthropic 连续调价 严重算力荒再现
算力涨价潮正在席卷全球，海内外巨头又开始了新一轮调价。阿里云四天连发三条产品涨价公告，调整百炼部分模型单元服务价格及 DataWorks 部分 API 免费额度。Anthropic 调整了旗下企业级产品 Claude Enterprise 订阅模式，从每月每用户最高 200 美元的固定费用，转变为按照实际算力消耗收费，此外用户每月还需要固定缴纳 20 美元——软件授权谈判协助公司 Redress Compliance 联合创始人 Fredrik Filipsson 表示，对于重度用户而言，新定价可能导致成本翻倍甚至三倍。进入 2026 年以来，全球算力需求快速增长，增长曲线陡峭，4 月全球最大的 API 聚合平台 OpenRouter 平台周度累计 Token 消耗量较一年前提升约 7-8 倍，其中国产大模型是爆发主驱力。中信证券指出，Token 调用量井喷带来的是算力需求极大爆发，与此同时供给侧受到各类硬约束短期边际增量有限，目前国内外均出现了严重的算力荒。
