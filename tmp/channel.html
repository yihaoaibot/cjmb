加拿大央行：金融体系稳健，但脆弱性有所增加
金十数据 5 月 28 日讯，加拿大央行周四表示，尽管身处高度动荡的经济和地缘政治环境中，加拿大金融体系目前状况良好，但其脆弱性正在增加。加拿大央行高级副行长卡罗琳·罗杰斯表示：“我们的总体看法是，加拿大金融体系依然具备抵御冲击的良好能力。”“然而，金融体系的某些部分，其脆弱性已有所增加。”罗杰斯指出，股市估值过高、企业债务水平上升，以及对冲基金为购买主权债务而借入的资金规模扩大，均属于当前金融体系面临的脆弱性因素。她表示，虽然这些风险若单独来看尚属可控，但日益动荡的经济和地缘政治环境可能会引发连锁反应，从而酿成大患。

微软下周将推出全新代码模型
微软将于下周在旧金山举办的年度开发者大会 Build 上，正式发布多款自研人工智能模型，此番举措意在争取开发者群体的认可。据了解相关规划的知情人士透露，微软计划推出一款代码专用模型，以此提升旗下代码助手 GitHub Copilot 的竞争力。这款工具曾在 AI 代码辅助领域占据先发优势，如今市场份额却被 Cursor 与 Claude Code 不断蚕食。该知情人士还表示，微软还将发布多款不同参数规格的全新模型，分别专攻语音转写、逻辑推理、语音处理及图像生成等场景。这一系列新模型，是在微软今年早些时候对外展示的自研模型基础上迭代升级而来。正如斯蒂芬妮在周三的报道中详细解读的那样，语音处理与语音转写功能目前已成为开发者关注的热门方向。微软官方发言人对此不予置评。（环球市场播报）
