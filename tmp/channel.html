中国移动邱宝华：未来 3-5 年将投入百亿级 Token 生态资源
2026 移动云大会上，中国移动市场经营部总经理邱宝华表示，未来，中国移动将围绕构建 Token 运营体系开展四大行动，即 Token 汇聚行动、平台筑基行动、消费激活行动、价值共赢行动。近期，中国移动将重点聚焦大模型深度合作，打造优质 Token 供给矩阵，核心通过三个路径：一是闭源模型直接接入，通过对接行业龙头大模型，快速丰富模型供给，满足用户多样化调用需求；二是开源模型自有部署，依托自有算力部署、优化开源模型，强化自主可控的模型供给能力；三是优质模型协同共建，联合厂家部署优质闭源模型，深度适配定制化产品需求，夯实 Token 高效生产的核心能力基础。未来 3—5 年，中国移动将投入百亿级 Token 生态资源，建设千亿级算力基础设施，携手共创万亿级 AI 产业价值。
